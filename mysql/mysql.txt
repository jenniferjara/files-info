** Import file.sql **
---------------------

* inject script

Terminal:
	mysql -u <username> -p <databasename> < <filename.sql>

Example:
	mysql -u root -p wp_users_DB < dump.sql

* Login to your MYSQL console.
	mysql -u root -p

	use <name_of_your_database>;
	source <path_of_your_.sql_file>

Example:

	use wp_users_DB;
	source /var/www/project_a/dump.sql;
